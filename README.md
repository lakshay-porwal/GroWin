# GroWin — Invest Smart. Spend Wise.

> Built by Students, for Students.

GroWin is a React Native mobile app that helps students track expenses and start investing — without the complexity of traditional finance apps. No stock trading, no jargon. Just Mutual Funds, SIPs, and ETFs, designed for beginners.

## About

Most investing apps are built for experienced traders. GroWin is different.

It focuses exclusively on the safest, most beginner-friendly investment instruments — Mutual Funds, SIPs, and ETFs — while also helping students understand and control their daily spending. The result is one app that builds both habits at once: mindful spending and disciplined investing.

---

## Features

### Expense Tracker
- Log daily expenses with smart categories
- Monthly analytics and spending breakdowns
- Visual insights to understand where your money goes

### Investment Module
- Invest in Mutual Funds, SIPs, and ETFs
- Beginner-friendly interface with no technical charts
- Guided onboarding for first-time investors

### AI Financial Assistant
- Risk profiling: Low / Medium / High
- Personalized fund recommendations
- Smart insights driven by your actual spending behavior

### Goals & Savings
- Create financial goals — Laptop, Trip, Emergency Fund, anything
- Track progress visually over time
- AI-suggested monthly investment amounts to hit your targets

### Wallet System
- Add money via UPI or mock wallet (demo mode)
- Invest directly from your in-app wallet
- Full transaction history at a glance

### Smart Alerts
- SIP payment reminders so you never miss a date
- Budget warnings before you overspend
- Weekly financial health summaries

## App Flow

```
Sign Up          →   OTP-based authentication
Risk Profiling   →   AI-driven questionnaire to understand your comfort level
Dashboard        →   Balance overview + personalized insights
Track Expenses   →   Log and categorize your daily spending
Fund Wallet      →   Add money via UPI or mock payment
Invest           →   Start a SIP, pick a Mutual Fund, or buy an ETF
Get Reports      →   Weekly AI summaries and financial health updates
```

## Getting Started

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/growin.git
cd growin

# 2. Install dependencies
npm install

# 3. Start the app
npx expo start
```

**Prerequisites:** [Node.js](https://nodejs.org/) and [Expo CLI](https://docs.expo.dev/get-started/installation/) must be installed on your machine.

## Limitations

GroWin is intentionally scoped. It does **not** include:

- Stock trading or derivatives
- Complex financial instruments or advanced charting
- Portfolio rebalancing or tax optimization tools

This keeps the experience safe, simple, and focused for beginners.
